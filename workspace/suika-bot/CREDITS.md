## Technologies & Libraries Used

### Core Framework
- **[Telegraf](https://telegraf.js.org/)** - Official Telegram Bot API framework for Node.js
- **Node.js** - JavaScript runtime environment

### Database
- **[Mongoose](https://mongoosejs.com/)** - MongoDB object modeling
- **MongoDB** - NoSQL database

### HTTP & Web
- **[Axios](https://axios-http.com/)** - Promise-based HTTP client
- **[Express](https://expressjs.com/)** - Web application framework (optional)

### Security & Utilities
- **[bcryptjs](https://github.com/dcodeIO/bcrypt.js)** - Password hashing
- **[dotenv](https://github.com/motdotla/dotenv)** - Environment variable management
- **[lodash](https://lodash.com/)** - Utility library
- **[fs-extra](https://github.com/jprichardson/node-fs-extra)** - File system enhancements

### Data Processing
- **[Cheerio](https://cheerio.js.org/)** - jQuery-like syntax for parsing HTML
- **[moment-timezone](https://momentjs.com/timezone/)** - Timezone handling

### Development
- **[Ora](https://github.com/sindresorhus/ora)** - Elegant terminal spinners
